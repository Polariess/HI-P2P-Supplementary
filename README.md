# HI-P2P-Supplementary

This repository contains the supplementary materials for the paper "HI-P2P: Human-Instructed Paper-to-Poster Generation Framework", including:
﻿
- `prompt_templates/prompt.pdf`: Complete prompt templates used in the experiments
- `auto_examples/auto.pdf`: Output examples generated by the automatic method
- `user_instructions_examples/HI.pdf`: Demonstration of human–computer interaction examples
﻿
These materials are provided to support the reproducibility and verifiability of the reported results.
